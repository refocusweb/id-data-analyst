Materi Kursus

Yang akan kamu pelajari:

  • Siapakah data analyst dan apa yang mereka kerjakan
  • Perbedaan antara data analyst, data engineer, data scientist, dan system analyst
  • Perusahaan apa yang membutuhkan data analyst
  • Cara memulai karier di data analytics dan bagaimana prospeknya (jenjang karier)
Yang akan kamu pelajari:

  • Menggunakan fungsi dan formula Spreadsheet
  • Menganalisis data dengan Pivot Table dan VLOOKUP
  • Membuat laporan analitis sederhana di Spreadsheet
  • Menggunakan teknik rendering dasar
Yang akan kamu pelajari:

  • Memahami struktur database
  • Menulis query database menggunakan SQL
  • Mengurai data dari banyak tabel menggunakan JOIN
  • Bekerja dengan berbagai jenis data
Yang akan kamu pelajari:

  • Mengimpor berbagai jenis dataset untuk Power BI
  • Menggunakan Power Query Editor dalam mempersiapkan data untuk visualisasi
  • Memperagakan data dengan membuat relasi antar tabel
  • Menghitung metrik bisnis menggunakan DAX
Yang akan kamu pelajari:

  • Menerapkan teknik visualisasi dasar dan lanjutan untuk dashboard
  • Menggunakan Row-Level Security dalam menyaring data untuk berbagai pemegang kepentingan
  • Memberikan wawasan dan rekomendasi bisnis berdasarkan data
Di akhir modul, kamu akan bisa:

  • Menulis perintah dasar pada Python
  • Mengidentifikasi berbagai jenis nilai dan mengubahnya
  • Menerapkan control statement (pernyataan kontrol) pada objek Python
  • Menggunakan fungsi dan operator dasar untuk memecahkan masalah bisnis sederhana

FAQ

Kamu tidak harus ahli di bidang Data Analytics untuk mendaftar di kursus ini. Namun, jika kamu memiliki pemahaman dasar tentang Excel atau SQL, itu akan bermanfaat.
Kamu dapat belajar kapan pun kamu mau setelah kursus dimulai. Ingatlah bahwa kamu akan mendapatkan akses ke setiap modul pada waktu yang telah ditentukan.
Kami mendorong para siswa kami untuk menyempatkan diri mengerjakan tugas selama 5–10 jam per minggu untuk menyelesaikan kursus. Kurang lebih terdapat 100 jam pembelajaran. Kamu juga harus mengerjakan sekitar 6 project, baik secara individual maupun kelompok.
Tidak, kamu harus menyelesaikan modul secara berurutan dan mengerjakan semua tugas sebelum melanjutkan ke modul berikutnya.
Setelah kursus dimulai, kami akan membuat channel di messenger yang beranggotakan semua siswa kursus. Kamu akan dapat berkomunikasi dengan orang-orang yang memiliki tujuan sama denganmu. Kamu juga dapat mengajukan pertanyaan kepada mentor dan community manager melalui grup messenger tersebut.
